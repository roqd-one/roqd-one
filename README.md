# Ro

> Apple apps, SaaS tools, and desktop products.

![Ro portfolio cover](assets/cover.png)

I build Apple apps, SaaS tools, and focused desktop products.

This GitHub profile is used as a public portfolio and documentation hub for the products I build. Most repositories here are public showcase repos: product pages with documentation, screenshots, links, and lightweight release notes rather than public source code.

## Main projects

### qdBox
Finance and operations dashboard for indie developers and SaaS founders. Tracks revenue, MRR, subscriptions, projects, tasks, domains, and infrastructure.  
**Links:** [Repo](https://github.com/roqd-one/qdbox-app) · [Website](https://qdbox.one/) · [App Store](https://apps.apple.com/es/app/qdbox-projects-revenue/id6758437065)

### AirMQTT
Native MQTT client for macOS, iPhone, and iPad. Built for monitoring topics, publishing messages, filtering streams, and fast MQTT workflows.  
**Links:** [Repo](https://github.com/roqd-one/airmqtt-app) · [Website](https://airmqtt.online/) · [App Store](https://apps.apple.com/es/app/airmqtt/id6745257530)

### PostFox
Autoposting and content workflows for social media. Designed to reduce routine publishing work and support more structured content operations.  
**Links:** [Repo](https://github.com/roqd-one/postfox) · [Website](https://postfox.app/) · [App Store](https://apps.apple.com/es/app/postfox/id6754407815)

### AlcoList
Alcohol tracker focused on limits, clarity, and better mornings.  
**Links:** [Repo](https://github.com/roqd-one/alcolist) · [Website](https://alcolist.online/) · [App Store](https://apps.apple.com/es/app/alcolist-alcohol-tracker/id6756630744)

### NowAgo
Habit and quit tracking for Apple devices, built around a simple and calm workflow.  
**Links:** [Repo](https://github.com/roqd-one/nowago) · [Website](https://nowago.app/) · [App Store](https://apps.apple.com/es/app/nowago-habit-quit-tracker/id6744716317)

### OrbiDeck
Minimalistic productivity app with tasks, notes, and projects.  
**Links:** [Repo](https://github.com/roqd-one/orbideck) · [Website](https://orbideck.com/) · [App Store](https://apps.apple.com/es/app/orbideck-to-do-list-notes/id6624294870)

### LumiGap
macOS tool for poker research and analysis.  
**Links:** [Repo](https://github.com/roqd-one/lumigap-poker-ai) · [Website](https://lumigap.com/en)
